# Моё портфолио 
## **Проект**: Интернет-магазин светового оборудования intersvetneo.ru 
По данному проекту я  составил тест-план, чек-листы, тест-кейсы, баг-репорты. 
____
 
### Инструменты использованные при подготовке данного портфолио:
* **Testlink** 
* **Redmine**
* **MYSQL WORKBENCH** 
* **POSTMAN**
______
